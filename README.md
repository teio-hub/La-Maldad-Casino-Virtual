
# Casino La Maldad ♣️

Casino La Maldad es un proyecto simple y divertido escrito en Python, que simula algunos de los juegos más populares de un casino.
Este proyecto busca recrear la experiencia de un casino clásico desde la consola, en la cual puedes registrarte y dar el valor de tu dinero.
Ve al Cashier's Cage para intercambiar dinero por fichas y viceversa.
Experimenta 8 grandes juegos de casino y prueba tu suerte!

Los juegos incluidos son:
- Blackjack
- Ruleta
- Tragamonedas
- Poker
- Baccarat
- Dados
- Bingo
- Carreras de caballos
